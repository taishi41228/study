## JavaScriptの始め方

```
<!DOCTYPE html>
<html lang="ja">
    <head>
    <meta charset="UTF-8">
    <title>JavaScript</title>
    <script src="JavaScriptのソースファイル"></script>
    </head>
    <body>  
    </body>
</html>
```

```
document.write('Hello World!');
```

文字列およびHTMLで記述された文書をブラウザに表示します。

構文

```
document.write(markup);
```

markup: 文字列およびHTMLで記述された文書

## 簡単な計算

記号 | 説明
------------ | -------------
+ | 足し算
- | 引き算
* | 掛け算
/ | 割り算
% | 余り

2+3
5-4
3*9
8/4
12%5
